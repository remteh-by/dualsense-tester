# DualSense Tester 🔵

Android-приложение для тестирования контроллера **PlayStation DualSense (PS5)**.

## Функции

- 🎮 Тестирование всех кнопок (✕ ○ □ △, L1/R1, L2/R2, Options, Create, PS, Touchpad)
- 🕹️ Визуализация стиков с настройкой мёртвой зоны
- 🎯 Калибровка — min/max отслеживание + сброс
- 📊 Отображение триггеров L2/R2 в реальном времени
- 🔊 Тестирование вибрации геймпада (короткая, средняя, длинная, паттерн)
- ⏱ Время удержания кнопок + счётчик нажатий
- 🐛 Raw Data дебаг для всех осей

## Установка

Скачайте APK из раздела [Releases](https://github.com/remteh-by/dualsense-tester/releases).

## Разработка

```bash
# Сборка debug APK
./gradlew assembleDebug

# Сборка release APK + AAB
./gradlew assembleRelease bundleRelease
```
